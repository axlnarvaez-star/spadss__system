# Supplier Performance Analysis and Decision Support System (SPADSS)

## Overview
The Supplier Performance Analysis and Decision Support System (SPADSS) is a web-based application designed to evaluate supplier performance using a weighted scoring model. The system helps organizations analyze supplier reliability, delivery performance, product quality, and pricing efficiency.

## Features
- Supplier performance evaluation
- Weighted scoring model
- Risk classification (Low, Moderate, High)
- Performance trend analysis
- Dashboard analytics
- User role management (Admin, Evaluator, Viewer)
- Email notification for account approval
- Export reports (PDF and CSV)

## Scoring Model
The system calculates supplier performance using the following formula:

Delivery – 40%  
Quality – 30%  
Price – 20%  
Reliability – 10%
